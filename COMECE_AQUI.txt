🚀 INICIAR RAPIDAMENTE:

   1. Clique duas vezes em: MENU.bat
   2. Escolha opção [1] - Iniciar Servidor Local
   3. Aguarde carregar (pode demorar na primeira vez)
   4. Acesse: http://localhost:5000
   5. Login: admin / admin123

════════════════════════════════════════════════════════════════

📱 TESTAR NO CELULAR:

   1. Inicie o servidor (MENU.bat → opção [1])
   2. Descubra seu IP (MENU.bat → opção [4])
   3. No celular, acesse: http://SEU_IP:5000
   4. Use as mesmas credenciais (admin/admin123)

   ⚠️ IMPORTANTE: Celular e PC na mesma rede Wi-Fi!

🐛 PROBLEMAS COMUNS:

   ❌ "dotnet: command not found"
      → Instale o .NET 8.0 SDK

   ❌ "Cannot connect to SQL Server"
      → Execute: testar-conexao-sql.bat
      → Verifique firewall e credenciais

   ❌ "Porta 5000 já está em uso"
      → Execute: parar-servidor.bat

   ❌ Servidor não responde
      → Feche tudo e reabra MENU.bat
      → Escolha opção [5] para limpar, depois [1] para iniciar

☁️ DEPLOY NO RAILWAY:

   1. npm i -g @railway/cli
   2. railway login
   3. railway init
   4. Configure variáveis de ambiente no painel
   5. railway up

   Detalhes completos em: GUIA_RAPIDO.md
